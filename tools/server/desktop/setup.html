🦙 alpacabitollama Setup

Welcome to alpacabitollama! Before you can start chatting, you need to download AI models.

  1. Right-click the system tray icon in your taskbar
  2. Select "Download Models" from the menu
  3. Wait for the models to download (~10GB total)
  4. Once downloaded, select "Start Server" from the tray menu
  5. The web interface will load automatically
