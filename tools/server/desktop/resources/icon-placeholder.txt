# Icon Placeholder

This directory should contain:
- icon.png - For macOS/Linux (256x256 or larger)
- icon.ico - For Windows (256x256 or larger)

To create proper icons:
1. Create a square PNG image at 256x256 or larger
2. Convert to ICO for Windows using tools like:
   - https://icoconvert.com/
   - ImageMagick: convert icon.png -define icon:auto-resize=256,128,96,64,48,32,16 icon.ico
3. Place both files in this directory

For now, the app will use a default empty icon if these files are missing.
